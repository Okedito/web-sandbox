Forms

Forms (excluding the appointment scheduling form, if used) are simple HTML forms: formatting will be applied using CSS styles on the front end, within the chosen technical context (available for Markerting Hub Pro and above).

[HubSpot's documentation provides more information on this topic.](https://knowledge.hubspot.com/forms/how-can-i-share-a-hubspot-form-if-im-using-an-external-site){:target="_blank"}

Integrating a HubSpot form on an external webpage is done by simply pasting the provided code into the div where the form should appear.

{% else %}

Formulaires

Les formulaires (hormis le formulaire de prise de RDV s'il est utilisé) sont des formulaires HTML simples : la mise en forme se fera au moyen de l'application de styles CSS au niveau du front end, dans le contexte technique choisi.

[La documentation de HubSpot donne plus d'informations sur ce sujet.](https://knowledge.hubspot.com/fr/forms/how-can-i-share-a-hubspot-form-if-im-using-an-external-site){:target="_blank"}

L'intégration d'un formulaire se fait par le simple dépôt du code fourni dans la div où doit apparaître le formulaire en question.
